MinGW
=====

mingw 4.8.1 folder to build whitecat with whitecat librairies

If you use the project whitecat_crossplateform\IDE\CodeBlocks\whitecatMinGW\Whitecat_Crossplatform\Whitecat_Crossplatform.cbp or whitecat_crossplateform\IDE\DEV-C++\WhiteCat_crossplateform.dev in the whitecat_crossplateform repository, you must: 
-or compile the libsource yourself 
-or use and uncompress this folder with 7zip into C:/ to be able to compile.

If you want compiling with your own version of MinGW and a GCC4.8.1 without compiling the library,
use simply the projects WhiteCat_crossplateform_with_relativ_lib.dev or Whitecat_precompiled_relativ_lib.cbp, set the compiler setting of your IDE with GCC and compile.
